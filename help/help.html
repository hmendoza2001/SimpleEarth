Simple Earth User’s Manual

 

What is Simple Earth?

Simple Earth is an open source, cross-platform virtual globe application. Its capabilities are:

-Display entire globe in 3D

-Easy navigation and controls

-Easy to load your own maps and icons

-Display high resolution satellite imagery

-Measure distances

-Emplace paths and volumes and save/load to/from file

 

How to Navigate

Simple Earth has two main navigation modes, orbit and perspective. The default navigation is orbit, where the camera is always pointing towards the center of the earth and its up vector points north. In perspective mode, the camera automatically “tilts” while getting close to the ground and keeps its up vector perpendicular to the ground. Under Perspective Mode, the user can tilt and yaw the camera. To enable Perspective Mode, select it from the toolbar options. The controls are as follows:

Navigate Left-Right-Up-Down                                                                       Arrow Keys, Mouse Drag with Left Mouse Button

Zoom In/Out (Change camera altitude in Perspective Mode)                +/- Keys, Mouse Wheel

Yaw (Only in Perspective Mode)                                                                  Q/E Keys, Mouse Drag with Middle Mouse Button

Tilt (Only in Perspective Mode)                                                                    W/S Keys

 

Tools

Simple Earth includes the following tools:

-Path Tool – Enables the user to plot paths on terrain and save them to file

-Volume Tool – Enables the user to emplace different types of volumes and save them to file

-Measuring Tool – Enables the user to measure distances on the globe

To enable a tool, you need to select it from the toolbar. All tool functionality is through the use of the right mouse button.

 

Menus

-File – Open or save the different file formats supported by Simple Earth

-View – Show/Hide different elements of the system

-Window – Open the different windows available in the system

-Help – Access to system information and help documentation

 

Windows

-Places – Allows the user to save the current camera position for later rapid navigation

-Track Info – Displays the list of all tracks on the left side. After the user selects a particular track, information about that track is displayed on the right side. You can bring up this window by selecting the menu option or by right clicking on a track.

-Paths and Volumes – Enables the user to show/hide current paths or volumes, to change their properties, or remove them from the list
